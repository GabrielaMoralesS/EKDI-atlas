# Validation

## Validation Needs

EKDI requires scientific validation before use as a formal conservation decision tool.

Recommended validation steps:

- Sensitivity analysis for EKDI weights.
- Expert review by botanists and ecologists.
- Herbarium and voucher review.
- Coordinate uncertainty review.
- Comparison with known survey sites.
- Comparison with recent independent botanical inventories.
- Review of regional land-cover uncertainty.

## GBIF Occurrence Data Limitations

GBIF-mediated records can include taxonomic changes, coordinate uncertainty, uneven sampling effort and incomplete dates.

EKDI does not judge GBIF data as wrong. It uses open records as evidence that should be interpreted carefully.

## Land-Cover Data Limitations

Land-cover products may contain classification error, temporal mismatch and resolution limits.

Post-record forest loss should be interpreted as a planning signal, not as a definitive site-level habitat assessment.

## Prototype Validation Status

The current Mata Atlântica dashboard is an advanced prototype.

It is suitable for demonstration, expert feedback and method discussion. It is not a final scientific publication.

## Recommended Review Questions

- Are EKDI weights appropriate for the Mata Atlântica case study?
- Are priority categories ecologically meaningful?
- Are Knowledge Ghosts and Rediscovery Candidates framed cautiously?
- Which voucher or herbarium sources should be added?
- Which field validation steps are required before publication?
