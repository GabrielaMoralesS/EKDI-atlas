# Presentation Outline

## 1. Problem

Biodiversity records remain valuable, but landscapes may change after records are collected.

## 2. EKDI Concept

An atlas for ecological memory and field-verification priorities.

## 3. Data Sources

GBIF records, biome boundary, land-cover change, grid and optional botanical enrichment.

## 4. Method

Sampling Antiquity, Post-Record Forest Loss and Richness Deficit combined into EKDI.

## 5. Demo Flow

Intro, priority map, selected-cell evidence, Botanical Evidence Summary and Expedition Planner.

## 6. Mata Atlântica Case Study

Real cleaned dashboard data for an Atlantic Forest prototype.

## 7. Botanical Value

Plant Candidates, Knowledge Ghosts, Rediscovery hypotheses and voucher review priorities.

## 8. Reproducibility

Static dashboard files, documented inputs and build script.

## 9. Limitations

Prototype status, data uncertainty, need for expert validation and field verification.

## 10. Next Steps

Professor feedback, GitHub publication, deployment, demo video and GBIF submission review.
